# Style Guide

This page defines the content and style standards for this wiki.

## Naming Conventions

All page titles should be in Title Case.

## Using Templates (MDX Components)

To ensure consistency, use the `<Callout>` component for important notes.

<Callout type="info">
  This is an informational callout.
</Callout>

<Callout type="warning">
  This is a warning callout.
</Callout>

<Callout type="error">
  This is an error callout.
</Callout>
